# TP – L'Apprenti Ordonnateur

## Description
Dans ce projet, l'apprenti ordonnateur doit réaligner des cristaux dans des temples après une tempête magique. Chaque temple a une couleur et contient un cristal pouvant être déplacé. L'apprenti peut transporter un seul cristal à la fois et se déplace pas à pas sur une grille.

## Fonctionnalités
- Lecture de scénarios depuis fichier texte
- Déplacement de l'apprenti et échange de cristaux
- Algorithmes de tri pour guider l'apprenti (niveau 1)
- Heuristiques ou solution optimale pour minimiser les déplacements (niveau 2 & 3)
- Interface graphique pour visualiser temples, cristaux et déplacement de l'apprenti
- Compteur de déplacements total

## Exécution
Lancer le programme principal via l'IDE Java ou la console.  
Charger un scénario et utiliser les boutons pour déplacer l'apprenti ou exécuter les algorithmes.

## Objectifs pédagogiques
- Manipulation de grilles et positions sur 2D
- Implémentation et comparaison d’algorithmes (tri, heuristiques)
- Gestion des états et visualisation graphique
- Optimisation de séquences d’actions dans un problème combinatoire
